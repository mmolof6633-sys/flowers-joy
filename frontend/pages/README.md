# Папка pages

Эта папка необходима для совместной работы Next.js и Feature-Sliced Design (FSD).

Next.js пытается использовать `src/pages` в качестве Pages Router, даже если вы используете App Router, что приводит к ошибкам при сборке проекта. Пустая папка `pages` в корне проекта предотвращает эту проблему.

Фактические страницы приложения находятся в `src/pages` (FSD-структура) и реэкспортируются в `app/` (Next.js App Router).
